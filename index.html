Bem-vindo à documentação da linguagem Tucano!

Tucano é um ecossistema digital criado para ensinar programação a jovens brasileiros, sendo uma linguagem de programação simples, intuitiva e escrita em português. Nesta documentação, você encontrará tudo o que precisa para começar a programar em Tucano, os conceitos básicos, exemplos práticos e referências avançadas.

Explore os recursos e ferramentas que o Tucano oferece para maximizar sua produtividade e criar soluções poderosas. Aproveite e deixe o seu feedback na nossa plataforma de ensino.

O ecossistema Tucano consiste de:
Uma plataforma para criação e compartilhamento de projetos escritos em Tucano. Acesse o TucanoPlay
Uma plataforma de cursos voltados para ensinar programação mais avançada à crianças. Acesse o Aprenda Tucano
Uma documentação interativa online. ← Você está aqui

Início

Tucano é uma linguagem de programação orientada a objetos bla bla bla
Para começar, computadores seguem algoritmos, e são esses algoritmos que determinam o que o computador fará, e computadores seguem esses algoritmos de forma sequencial, ou seja, uma linha de código após a outra. Para começar então, você precisa entender o que são variáveis, operadores, laços de repetição e funções. Para o primeiro contato, você pode começar a escrever um programa simples que imprime "Olá, mundo!" na tela.

escreval("Olá, mundo!")

Após escrever esse código, você pode notar que o computador executou a instrução e imprimiu "Olá, mundo!" na tela. Isso é o básico de programação: escrever instruções que o computador executa. A partir disso, você pode montar algoritmos mais complexos, utilizando variáveis para armazenar dados, operadores para realizar cálculos e laços de repetição para executar ações repetidamente. Nessa documentação, você encontrará exemplos e explicações detalhadas sobre cada um desses conceitos.

Ler e Escrever

A linguagem Tucano permite que você escreva e leia dados de forma simples. Você pode usar a função escreval() para imprimir mensagens na tela e a função leia() para receber entradas do usuário. Veja um exemplo:

escreval("Digite seu nome:")
caractere nome = leia()
escreval("Olá, ", nome, "!")

Você pode perceber que o código acima lê o nome do usuário, mas ele faz mais que isso, ele armazena o nome em uma variável chamada nome e depois imprime uma mensagem personalizada. Essa é a essência da programação: manipular dados e interagir com o usuário.
